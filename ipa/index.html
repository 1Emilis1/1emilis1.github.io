Why use iPATool-EZ?

Simple Interface

Designed to be much easier to use than standard CLI tools.

Multi-Account

Manage and switch between up to 20 Apple IDs seamlessly.

2FA Support

Full support for Two-Factor Authentication during login.

Delisted Apps

Download apps that are no longer available on the public App Store.

Installation

Windows

  1. Install Python 3+.
  2. Download the latest release from GitHub.
  3. Run install.bat.
  4. Run ipatool-ez.bat to start.

macOS / Linux

  1. Install Python 3+.
  2. Download the latest release.
  3. Run install.sh in your terminal.
  4. Run ipatool-ez.sh to start.

How to Use

After running the tool, follow the on-screen prompts to:

  • Create Account: Link your Apple ID safely.
  • Search: Find the app you want to download.
  • Download: Get the encrypted .ipa file directly to your computer.

Note: You must own the app on your Apple ID to download it!
